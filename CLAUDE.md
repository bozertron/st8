## Entry-point chain

```
start.js (repo root, 148 LOC)        ← process entry: parses argv, installs deps, spawns backend
   │   spawn('node', ['src/core/server/main.js', targetDir, '--port', N, '--serve'])
   ▼
src/core/server/main.js (453 LOC)    ← backend bootstrap: wires indexer + persistence
   │                                    + manifest + watcher + St8Server + hookRegistry
   │   new St8Server(...).listen(...)
   ▼
src/core/server/app.js (2264 LOC)    ← HTTP API surface (St8Server class)
                                       All /api/* routes live here.
                                       This is where most edits land.
```

**`app.js` is the HTTP layer, NOT the entry point.** `start.js` → `main.js` → `app.js`.

## Hooks / signal flow

Canonical hook constants in `src/core/hook-registry.js`:
- `INDEX_START`, `INDEX_COMPLETE`
- `FILE_INDEXED`, `FILE_BEFORE_CHANGE`, `FILE_AFTER_CHANGE`
- `LIFECYCLE_TRANSITION`, `COMMIT_RECORDED`
- `PRD_GENERATE`, `TICKET_CREATED`

All have at least one publisher. Subscribers register in `src/core/hooks/default-subscribers.js` with explicit `source` tags and priorities (lower priority runs first). Per-subscriber convention: **DRY + wrap** — each subscriber wraps its own body in try/catch with `[st8] <source> failed: <msg>` log; registry-level catch is the safety net.

## Active sprint protocol

The 23-wave debugging sprint is documented in `st8_bible.md` (batches 028+).

Per-wave structure:
- **Executor agent** — runs 5-ish tickets in a single concern. Reads `PRE_FLIGHT.md`, runs pre-flight, executes, commits per-ticket, updates `docs/_pending-tickets/<cluster>.json`.
- **Reviewer agent** — audits the executor's commits. Updates verdicts in JSON. Appends to `docs/_pending-tickets/<cluster>.review.md`. Last reviewer in cluster renames JSON to `.for-review.json`.
- **Watchdog Monitor** — alerts on stalls (>15 min uncommitted-without-mtime-update OR >20 min wall-clock-since-last-commit).
- **Hard rule**: no extras per wave. Flags surfaced by reviewers become their own sub-wave.
