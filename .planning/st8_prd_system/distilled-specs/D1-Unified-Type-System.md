## Abstract

This document defines the canonical TypeScript type system for the st8 PRD (Product Requirements Document) module. It merges concepts from six distinct lineages — Orchestr8 PrdGenerator/PrdEditor, TaskMaster parse-prd, SCOUT, MetaGPT write_prd, stereOS/actu8, and the st8 Identity System — into a single unified set of interfaces. Each type includes its lineage provenance, mapping notes to existing st8 artifacts, and design rationale.
